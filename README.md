# AliceSitesMonitoring
Jupyter Lab notebook to produce some summary plot.

Requisites:
- Jupyter Lab
- Docker

### Deploy InfluxDB
From the folder _influx_ run:

```
./run.sh
./create_db.sh
```

### Deploy Grafana

From the folder _grafana_ run:

```
./run.sh
```

### Start Jupyter Lab 
 
 From the local folder run:
 
 ```
 jupyter lab
 ```
